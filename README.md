# 目的

- 每日练习一条不一样的linux 命令.掌握 linux 系统下核心常用的命令。
- 创建一个linux技术和知识分享的圈子, 共同提高linux技术。
- 尽可能统一osx和linux下的命令，做到不重复学习，平台通用。
- 每周六练习linux命令, 周日分享linux系统知识
